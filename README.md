# 项目说明文档

## 一 运行说明
1. 将项目使用git clone到本地
2. 本地安装node环境
3. 执行 npm install 安装项目所需要的依赖
4. 执行 npm run server 进行实时热加载开发
5. 打包执行 npm start

## 三 项目技术栈
1. 采用**webpack**构建
2. 使用**react**框架
3. 使用**less**的css预编译技术
4. 使用**es6**进行代码编写
5. 阿里的**iconfnt**图标库


## 四 组件结构
1. 整个页面分为三个部分：顶部、中部、底部。
2. 顶部分为：登录，搜索，导航
3. 中部分为：广告横幅、名企采购展示、名企推荐爆品、最新采购情况、采购政策、行业资讯、答疑解惑
4. 底部分为：友情链接，公司名称和链接的
