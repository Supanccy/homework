# 项目说明文档
## 一采用技术
1. 采用**webpack**构建
2. 使用**react**框架
3. 使用**less**的css预编译技术
4. 使用**es6**进行代码编写
5. 阿里的**iconfnt**图标库


## 二组件结构
1. 整个页面分为三个部分：顶部、中部、底部。
2. 顶部分为：登录，搜索，导航
3. 中部分为：广告横幅、名企采购展示、名企推荐爆品、最新采购情况、采购政策、行业资讯、答疑解惑
4. 底部分为：友情链接，公司名称和链接的

## 在效果图基础上新增组件
1. 固定位置的搜索框
2. 固定位置的回到顶部的火箭图标
3. 大幅图片轮播功能组件
4. 登录区域的二级菜单
